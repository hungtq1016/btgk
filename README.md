# TranQuocHung_501210025

## Các công nghệ sử dụng

VueJS , Vuex , Json-server

## Cài đặt sau khi tải về
```sh
npm install
```
### Cài đặt json-server-auth
Cài đặt tại global

```sh
npm install -g json-server-auth
```

### Cài đặt fake back-end

```sh
npm run db
```

### Cài đặt xong bắt đầu chạy dự án

```sh
npm run dev
```

### Chạy thử dự án

```sh
npm run build
```
